## 功能概述

除简单播放外，还实现了：

1. 单曲循环、列表循环功能
2. 手动切换乐器界面与播放列表界面，可点击播放列表切歌
3. 简化了界面和操作逻辑，使之比起代码逻辑本身更符合人类的操作习惯
4. 使界面半透明，更具美感
5. 计算总时长和已播放时长，并实时更新标题，方便用户直观观察播放时间
6. 解决了原程序出现死锁问题的bug
7. 添加了任务栏显示播放进度和控制播放功能，进一步提升了美观性和实用性
